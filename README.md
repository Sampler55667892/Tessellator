"# Tessellator" 

次のサイトで提供されているサンプルコードを
少しだけ一般化したもの．

OpenGL Tessellation
http://www.songho.ca/opengl/gl_tessellation.html

OpenGL はバージョン 4 辺りから，
1つの三角形を複数三角形に分割する (DirectX 11 ライクの)
ダイナミックなテッセレーションが使えるようになっているが，
これは，Version 1.x 辺りから使えた Polygon Triangulation．

プログラムの情報量は，上記のサンプルコードよりも小さく，
誰でも再現できる程度のものである．
ツールとして使える形に少し直しただけだが，
便利そうなのでアップしておく．
